Cos'è il design responsive?

Il design responsive è un approccio alla progettazione di siti web che garantisce che i contenuti e il layout si adattino automaticamente a diverse dimensioni di schermo e dispositivi. Questo significa che un sito web è facilmente leggibile e navigabile sia su un computer desktop che su uno smartphone o tablet.

Ecco alcune caratteristiche principali del design responsive:

  • Adattabilità: Il layout cambia in base alle dimensioni dello schermo.
  • Fluidità: Gli elementi si ridimensionano proporzionalmente per riempire lo spazio disponibile.
  • Media Query: Regole CSS che definiscono stili specifici per diverse larghezze di schermo.

Un esempio di regola CSS con media query:

@media (max-width: 768px) {
    body {
        font-size: 14px; /* Ridimensiona il testo per schermi piccoli */
    }
}

Come funziona il menù responsive?

Un menù responsive consente agli utenti di navigare facilmente su dispositivi con schermi piccoli, come smartphone e tablet. La versione base di un menù responsive include:

  • Hamburger Menu: Un'icona (generalmente tre linee orizzontali) che, al clic, mostra o nasconde il menù.
  • Input Checkbox: Utilizzato per gestire l'apertura e la chiusura del menù.
  • Media Query: Nasconde il menù completo su schermi piccoli e visualizza solo l'icona hamburger.

Con il seguente CSS per nascondere e mostrare il menù:

.hamburger {
    display: none;
}

.nav-links {
    display: flex;
}

@media (max-width: 768px) {
    .hamburger {
        display: block;
    }

    .nav-links {
        display: none;
        flex-direction: column;
    }

    #menu-toggle:checked + .nav-links {
        display: flex;
    }
}
