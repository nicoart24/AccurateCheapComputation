Spiegazione: Come gestire le immagini in HTML

In HTML, il tag <img> viene utilizzato per inserire immagini in una pagina web. Questo tag ha alcuni attributi fondamentali:

  • src: Specifica il percorso del file immagine. Ad esempio: 
    <img src="img/ARTandSETUPacademy_logo.jpg" alt="Descrizione dell'immagine">
  • alt: Fornisce una descrizione testuale dell'immagine, utile per l'accessibilità e quando l'immagine non è disponibile.
  • class: Utilizzato per applicare stili specifici tramite il CSS, come il ridimensionamento per immagini responsive.

Per rendere le immagini responsive, è importante utilizzare il CSS. Ad esempio: 

/* Immagine a larghezza piena */
.full-width {
    width: 100%; /* L'immagine occupa tutta la larghezza */
    height: auto; /* Mantiene le proporzioni */
}

/* Due immagini affiancate */
.half-width {
    width: calc(50% - 10px); /* Ogni immagine occupa il 50% meno lo spazio tra loro */
    height: auto;
}

/* Tre immagini affiancate */
.third-width {
    width: calc(33.33% - 10px); /* Ogni immagine occupa un terzo della larghezza */
    height: auto;
}

In combinazione con le media query, possiamo adattare automaticamente il layout delle immagini per dispositivi con schermi più piccoli. Ad esempio: 

@media (max-width: 768px) {
    .half-width,
    .third-width {
        width: 100%; /* Le immagini si adattano a tutta la larghezza dello schermo */
    }
}
